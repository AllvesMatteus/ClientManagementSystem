AVISO IMPORTANTE - PROJETO FICTÍCIO

Este projeto é inteiramente fictício e foi desenvolvido exclusivamente para fins de treinamento nas tecnologias TypeScript e MySQL. Qualquer dado apresentado não representa informações reais de empresas ou clientes.

Todos os dados cadastrados no sistema devem ser fictícios. Não insira informações pessoais ou sensíveis, pois o banco de dados é público e aberto, sendo utilizado apenas para demonstração e estudos.

Caso o sistema apresente instabilidades ou indisponibilidade, isso pode ocorrer devido ao uso da versão gratuita do Supabase, que possui limitações de recursos e uptime.
